 # 🔍 Finder: Building Bridges in the Tech World. 
 ### 🌐 Full Stack Web Application Project

 **What is Finder?**

Finder is an innovative web application developed using React for the client side and Python for the server side. Designed to bridge the gap between companies and IT talent, it provides a dynamic platform for job seekers and employers to connect effortlessly. With a focus on the tech industry, Finder simplifies the job search and hiring process while prioritizing security, accessibility, and a seamless user experience.

The platform empowers users by offering tools to browse job postings, manage personal profiles, and apply for positions with ease. Candidates can explore opportunities tailored to their skills and interact directly with potential employers. For companies, Finder streamlines the recruitment process by enabling efficient vacancy management and organized handling of applications.

Finder is more than just a job board—it's a space where talent meets opportunity, fostering connections that drive careers and innovation forward.

## 💻 Technologies Applied
### 🎨 Front-end

- **React:** Library for building dynamic and responsive user interfaces.
- **JavaScript:** Main language for frontend logic.
- **Bootstrap 5:** CSS framework for styling and responsive design.
- **CSS 3:** Customization of styles to enhance visual appearance.

### 🔌 Back-end

- **Python:** Main language for backend logic.
- **Flask:** Framework for efficiently creating RESTful endpoints.
- **SQLAlchemy:** ORM for managing the database securely and efficiently.

## 📚 Applied Knowledge
### 📡 Global State Management with Context API (Flux)
- Implementation of a centralized system to manage the application's state.
- Improved consistency and data flow between components.
### ⛵ React Router for Navigation
- Usage of React Router to implement dynamic routes and secure access based on user state.
### 🔐 bcrypt for Encryption
- Robust security for storing passwords through encryption.
- Enhanced protection for sensitive data.
### ⚙️ Usage of Hooks and Custom Hooks
- Leveraging hooks like useState, useEffect, and useContext.
- Creation of custom hooks to encapsulate reusable logic.
### 🧩 Application of SOLID Principles:
- Adoption of software design principles to ensure code maintainability and scalability.
### 🔑 JWT (JSON Web Tokens):
- Secure authentication using tokens.
- Middleware implementation to protect sensitive routes.
### 📬 Postman:
- Tool for efficiently testing and documenting endpoints.
- Validation of HTTP responses and backend functionality tests.

> ## 🛠️ Installation:
> If you use Github Codespaces (recommended) or Gitpod this template will already come with Python, Node and the Posgres Database installed. If you are working locally make sure to install Python 3.10, Node
> 
> ### Back-End Manual Installation:
> - It is recomended to install the backend first, make sure you have Python 3.8, Pipenv and a database engine (Posgress recomended)
> 1. Install the python packages: `$ pipenv install`
> 2. Create a .env file based on the .env.example: `$ cp .env.example .env`

> ### Front-End Manual Installation:
> - Make sure you are using node version 14+ and that you have already successfully installed and runned the backend.
> 1. Install the packages: `$ npm install`
> 2. Start coding! start the webpack dev server `$ npm run start`
